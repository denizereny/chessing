Feature: adaptive-viewport-optimizer

Property 25: Resize Event Debouncing

Validates: Requirements 8.3

Property Under Test

For any rapid sequence of resize events (more than 10 events within 100ms), the number of layout recalculations should be significantly less than the number of events (at most 1-2 recalculations).

This property ensures that the ViewportAnalyzer properly debounces resize events to prevent excessive layout recalculations, which would degrade performance during window resizing.
